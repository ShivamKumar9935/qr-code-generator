📌 QR Code Generator (Python)

A lightweight Python script that generates a QR code for any URL you provide.

## 🌐 Live Demo
👉 https://qr-code-generator-owua.onrender.com/


🚀 Features

Generate QR codes from any text or URL

Saves the QR code as an image file

Simple and beginner-friendly

Cross-platform (Windows, macOS, Linux)

🛠️ Requirements

Python 3.x

qrcode library

Install dependencies:

pip install qrcode[pil]

📂 Project Structure
qr-code-generator/
│── qr_generator.py
│── output/
│   └── qrcode.png
│── requirements.txt
│── README.md

▶️ How to Run

Clone the repository:

git clone https://github.com/ShivamKumar9935/qr-code-generator.git


Navigate to the project folder:

cd qr-code-generator


Run the script:

python qr_generator.py


Enter a URL when prompted:

Enter the URL: https://youtube.com


The QR code image will be saved in the output/ folder.

📸 Example Output

A QR code image (qrcode.png) containing the entered URL.

💡 Use Cases

Sharing links quickly

Generating QR codes for events or posters

Learning Python basics and libraries
